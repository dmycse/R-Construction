# R-Construction - Company's Landing Page

## Overview

**R-Construction LP** - a sleek and professional landing page designed to highlight a construction team’s expertise and reliability. It showcases completed projects, the team’s skills, and testimonials from satisfied clients, emphasizing quality craftsmanship and attention to detail. Visitors can easily request a quote or consultation, making the path from interest to hiring simple and trustworthy.

---

## 🚀 Tech Stack
| Category   | Technologies & Libraries |
|-------------|---------------------------|
| **Framework** | [Astro v.5+](https://astro.build/) |
| **Styling**   | [TailwindCSS v.4+](https://tailwindcss.com/) |
| **Languages** | HTML, CSS, TypeScript |

---

## ⚙️ Installation & Local Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/dmycse/R-Construction.git
cd r-construction
npm install
```

Start the local development server:
```bash
npm run dev
```

Preview the production build locally:
```bash
npm run preview
```
